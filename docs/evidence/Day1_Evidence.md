## 项目规划

### 页面清单

| 页面 | 路由路径 | 功能说明 |
|------|---------|---------|
| 首页 | `/` | 商品列表展示、分类导航 |
| 商品详情 | `/product/:id` | 商品图片、描述、价格、联系卖家 |
| 商品发布 | `/product/new` | 发布新商品 |
| 搜索结果 | `/search` | 关键词搜索 + 分类/价格筛选 |
| 登录 | `/login` | 账号密码登录 |
| 注册 | `/register` | 新用户注册 |
| 个人中心 | `/profile` | 用户信息、我的发布、我的订单 |
| 购物车 | `/cart` | 购物车商品管理 |
| 订单列表 | `/orders` | 所有订单状态概览 |
| 订单详情 | `/order/:id` | 单个订单完整信息 |

### 功能模块

- **用户模块** — 注册/登录、个人信息管理、我的发布
- **商品模块** — 商品列表、商品详情、商品发布/编辑/下架
- **交易模块** — 购物车、订单创建、订单状态跟踪
- **搜索模块** — 关键词搜索、分类筛选、价格区间筛选

### 开发顺序

| 阶段 | 任务 |
|------|------|
| Phase 1 | 项目初始化（路由、Store、API 封装） |
| Phase 2 | 用户模块（注册/登录、Token、路由守卫） |
| Phase 3 | 商品展示（首页列表、商品详情） |
| Phase 4 | 商品发布（发布表单、图片上传） |
| Phase 5 | 搜索筛选（关键词 + 分类/价格） |
| Phase 6 | 购物车（加入、管理、结算） |
| Phase 7 | 订单管理（列表、详情、状态流转） |
| Phase 8 | 优化收尾（错误处理、UI 优化） |

### 开发重点

1. 用户认证体系 — Token 存储与路由守卫
2. 商品图片处理 — 上传预览与压缩
3. 搜索筛选性能 — 防抖与缓存
4. 订单状态管理 — 生命周期流转
5. 响应式布局 — 优先移动端体验

## 学习收获

- 理解了 Vue 3 项目的标准目录结构：`src/api`、`src/components`、`src/router`、`src/stores`、`src/views` 各自职责清晰
- 掌握了 `main.ts` 作为入口文件的作用：创建 Vue 实例 → 注册 Pinia → 注册 Router → 挂载 DOM
- 理解了 Vue Router 如何实现 SPA 页面导航，以及 HTML5 History 模式与 Hash 模式的区别
- 熟悉了项目的技术栈选型：Vue 3 + TypeScript + Vite + Pinia，是一套现代化的前端工程化方案
- 掌握了 SSH Key 生成和 GitHub 远程仓库配置流程
- 学会了如何进行项目规划，包括页面划分、功能模块拆解和开发优先级排列
